Was ist eigentlich RCS?

Was bringt RCS dem User?
  • Nachrichten werden mit Metadaten angereichert z.B. (GPS Karten / Flugtickets)
  • SMS Ersatz (ohne Kosten)
  • Erreichen von Menschen über Plattform / Netzgrenzen hinweg
  • File Transfer / Gruppenchat
  • Wachsendes Feature Set (z.B.Videotelefonie)
